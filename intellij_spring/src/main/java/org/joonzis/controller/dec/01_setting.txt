1. 개발 환경
	1) 언어 : JDK 8
		- https://www.oracle.com/java/technologies/javase-downloads.html
	2) 툴 : STS
		- 이클립스[Help] -> [Eclipse MarketPlace..]선택 -> Spring으로 검색
		-> Spring Tools 3 플러그인 설치
	3) 웹 서버 : Tomcat
		- http://tomcat.apache.org/download-90.cgi
	4) DataBase : Oracle(11g Express Edition)
		- https://www.oracle.com/database/technologies/xe-prior-releases.html
		- 데이터 학습용 
		
2. 롬복 (Lombok)
	- 코드량을 줄여주는 java 어노테이션 라이브러리
	- 여러 기능이 있지만 객체 클래스에 사용하여 getter/setter/toString/생성자등을 자동 생성
	1) 설치
		- https://projectlombok.org/download
		- jar파일 설치 후 이클립스 위치에 이동
		- 커맨드 창을 이용하여 해당 위치에서 lombok.jar 실행
			(1) 더블클릭하여 실행하는 경우
			(2) 더블클릭으로 압축이 풀리는 경우
				-> 커맨드창을 이용하여 실행
				- 롬복파일 위치에서 명령어 실행
