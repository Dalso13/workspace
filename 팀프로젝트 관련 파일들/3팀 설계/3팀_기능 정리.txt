1. 메인페이지
- 로그인 X : 로그인, 회원가입
- 로그인 O : 로그아웃, 마이페이지, 건의사항 등록
- 관리자 로그인 O : 로그아웃, 관리자 페이지
- 로고 클릭 시 새로고침
- 검색창 : 일단 나라만 검색 가능(시간 남으면 도시 검색했을 때 도시 데이터가 나오게)
- 세계지도 선택 : 대륙 줌인 > 세부 대륙 줌인 > 나라 리스트 나오게 > 나라 하나 클릭 시 게시글 이동

2. 유저가 건의사항 등록하는 페이지
- 로고 클릭 시 메인페이지로
- 건의사항 내용 입력
- 모든 데이터 입력 후 게시글 등록 버튼 클릭 시 "등록되었습니다." 알림 뜨고 메인 화면으로 이동

3. 나라 하나 선택 후
- 로고 클릭 시 메인페이지
- 검색창: 메인페이지랑 같은 검색창
- 카테고리 이동 가능 : 관광지, 맛집, 숙소
- 관리자가 등록한 나라 소개 정보
- 유저 게시글들 일부 보여줌
- 더보기 누르면 유저 게시글 목록 페이지로 이동

4. 숙소 카테고리 선택 시
- 로고 클릭 시 메인페이지
- 검색창: 메인페이지랑 같은 검색창
- 필터 : 유저 게시글처럼 나라 선택 가능, 카테고리 이동 가능
- 체크인, 체크아웃 날짜 캘린더 라이브러리 사용해서 선택
- 인원도 입력
=> 날짜, 인원 데이터 확인해서 매칭된 결과만 리스트에 뜨게
=> 맨 처음에는 모든 리스트가 순서대로 보이게(기본값)
- 호텔 리스트들(이미지 등 정보, 구글맵)
- 관리자로 로그인 했을 때 게시글 등록 버튼 활성화
- 페이징처리

5. 숙소 상세페이지
- 로고 클릭 시 메인페이지
- 검색창: 메인페이지랑 같은 검색창
- 이미지 보여줌
- 직전 페이지에서 날짜, 인원 선택하면 그대로 값 가져오고, 선택하지 않으면 상세 페이지에서도 선택 가능하게
- 만약 상세페이지에서 날짜, 인원 선택했는데 결제 못하는 숙소면 결제 버튼이 없어시고 품절되었습니다 라고 뜨게 하거나 애초에 날짜 선택할 때 못하게 막아두기
- 숙소 기본 정보 보여주기
- 숙소 주소를 기반으로 구글맵 보여주기
- 관리자로 로그인 했을 때 게시글 수정, 삭제 버튼 활성화
- 로그인한 사람만 결제 페이지로 넘어가게(결제 버튼)

6. 관리자가 숙소 등록하는 페이지
- 숙소 등록 입력
- 카테고리에 따라서 숙소, 맛집, 관광지 입력 폼 변경

7. 관리자가 숙소 수정하는 페이지
- 기존의 값 전부 가져온 상태에서 수정 가능하게

8. 맛집 카테고리 선택 시
- 필터 숙소랑 동일
- 상세 페이지 X
- 관리자만 게시글 등록 버튼 활성화
- 관리자만 게시글 수정, 삭제 가능하게
- 페이징
- 구글맵

9. 관리자가 맛집 등록하는 페이지
- 숙소랑 동일

10. 관리자가 맛집 수정하는 페이지
- 숙소랑 동일

11. 관광지 카테고리 선택 시
- 필터 숙소랑 동일
- 상세 페이지에서 수정, 삭제 가능하게
- 기능 숙소랑 비슷

12. 관광지 상세 페이지
- 관리자만 게시글 수정, 삭제 가능
- 숙소랑 기능 동일

13. 관광지 등록 페이지 
- 숙소랑 동일

14. 관광지 수정 페이지
- 숙소랑 동일

15. 결제 페이지 : 상세 페이지에서 결제 버튼 눌렀을 때 나오는 페이지
- 직전 페이지에 있던 데이터들 넘어와서 보여줌
- 구매자 정보 입력하는 기능(유저 정보 가져와서 read only)
- 결제 버튼 클릭 시 카카오페이로 이동

16. 유저 게시글 목록
- 필터 : 나라 선택 가능, 조회순으로 정렬, 좋아요순으로 정렬
- 페이징
- 로그인한 유저만 게시글 등록 버튼 활성화

17. 유저 게시글 상세페이지
- 본인 게시글만 수정, 삭제 가능
- 조회수 보여줌
- 로그인 한 사람만 1번 스크랩 가능
- 로그인 한 사람만 1번 좋아요 가능(다시 누르면 취소)
- 댓글 입력, 대댓글 입력 (각 좋아요)
- 게시글, 댓글, 대댓글 신고 가능
- 댓글 수 보여주기

18. 유저 게시글 등록
- 로그인 한 사람만 게시글 등록 가능
- 게시글 내용 입력할 때 이미지, 글 모두 태그 형태로 DB에 저장
- 카테고리(현재 선택한 나라 read only로 보여줌)

19. 유저 게시글 수정
- 등록이랑 동일
- 작성일 추가

20. 로그인
21. 회원가입
22. 아이디 찾기
23. 비밀번호 찾기
24. 비밀번호 변경

25. 마이페이지
- 내 정보 메뉴 클릭 시 내 정보 데이터 나옴 > 회원정보 수정 페이지로 이동 가능, 비밀번호 변경 페이지로 이동 가능
- 게시글 메뉴 클릭 시 내가 작성한 게시글 목록 보여줌
- 댓글 메뉴 클릭 시 내가 작성한 댓글 목록 보여줌
- 스크랩 메뉴 클릭 시 내가 스크랩한 게시글 목록 보여줌
- 결제 내역 메뉴 클릭 시 결제 내역 목록 보여줌
 
26. 관리자 페이지
- 관리자만 접근 가능한 페이지
- 게시글 신고 내용, 댓글 신고 내용, 건의사항 각 메뉴 선택 시 매칭된 데이터 목록 보여줌
- 게시글 신고 내용, 댓글 신고 내용 메뉴에서 처리하고 싶은 데이터 선택하면 실제 목록에서도 "숨김 처리 되었습니다 ~~ " 라고 보여줌(신고 받은 것만)
